# OpenFishPlugin

轻量的爬虫插件框架

```bash
ROOT
|-- android
|   |-- app
|   |-- gradle
|   |-- movie-runtime-android // Android版电影插件运行时
|   └─- runtime-android // Android版插件基础运行时
|-- doc
|-- javafx // JavaFx支持库
|   |-- app
|   |-- gradle
|   |-- movie-runtime-javafx // JavaFx版电影插件运行时
|   └─- runtime-javafx // JavaFx版插件基础运行时
|-- js
|   └─- src // JavaScript SDK
|-- jvm
|   |-- compiler // 插件编译器
|   |-- core // 核心库
|   |-- fdb // Fish Debug Bridge JVM支持
|   |-- gradle
|   |-- installer // 插件安装工具
|   |-- movie-runtime // 基础电影插件运行时
|   |-- packager // 插件打包工具
|   |-- runtime // 核心插件运行时
|   └─- tools //
|-- native
|   └─- fdb // Fish Debug Bridge
|-- sdk
|   |-- samples // 插件示例项目
|   └─- src // SDk
└-- tools
    |-- editor // 插件编辑器
    └─- editor2
```
